# EBIOS-Risk-Manager-Calculator
Calculateur automatisé pour l'analyse de risques selon la méthode EBIOS RM

Fonctionnalités :
- Module atelier 1 : Calcul de la gravité des événements redoutés
- Module atelier 2 : Matrice Sources de Risque × Objectifs Visés
- Module atelier 3 : Scoring des scénarios stratégiques
- Module atelier 4 : Priorisation des risques opérationnels
- Génération de cartographie des risques (heatmap)
- Export rapport EBIOS conforme ANSSI
